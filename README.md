# 💧 IoT Water Quality Monitoring System (ESP32)

An IoT-based water quality monitoring system built using ESP32 and standard sensors (pH, TDS, Turbidity), simulated in Wokwi. The system enables real-time observation of water parameters for environmental and smart monitoring applications.

---

## 🚀 Features

* 📡 Real-time monitoring of water quality parameters
* 🧪 pH level measurement
* 💧 TDS (Total Dissolved Solids) monitoring
* 🌫️ Turbidity detection
* ⚡ ESP32-based IoT architecture
* 🖥️ Simulation using Wokwi platform

---

## 🛠️ Tech Stack

* **Microcontroller:** ESP32
* **Programming Language:** Arduino (Embedded C/C++)
* **Simulation Platform:** Wokwi
* **Sensors:** pH, TDS, Turbidity

## ▶️ How to Run the Project

1. Open Wokwi simulator
2. Create a new project
3. Replace files with this repository files
4. Upload `diagram.json`
5. Run the simulation
6. Adjust sensor values using sliders
